# JavaScript DOM Manipulation

## Description
This project covers DOM manipulation techniques using JavaScript, including selecting elements, modifying content, handling events, and updating styles dynamically.

## Learning Objectives
- Understand the DOM structure
- Select and traverse DOM elements
- Modify element properties and attributes
- Handle user events
- Dynamically update HTML and CSS

## Requirements
- JavaScript ES6+
- Modern browser with developer tools
- Basic HTML/CSS knowledge

## Files
- `script.js` - Main JavaScript file
- `index.html` - HTML structure
- `style.css` - Styling

## Usage
```bash
# Open index.html in your browser
# Or use a local server:
python3 -m http.server 8000
```
