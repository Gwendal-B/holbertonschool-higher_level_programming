# Python - Object-Relational Mapping

## Description
This project covers Object-Relational Mapping (ORM) in Python, exploring how to interact with databases using Python objects instead of raw SQL queries.

## Learning Objectives
- Understand ORM concepts and benefits
- Work with database models and relationships
- Perform CRUD operations through ORM
- Handle database migrations and schema management

## Project Structure
All project files are organized in this directory:
- `models/` - Database model definitions
- `scripts/` - Utility and test scripts
- `tests/` - Unit tests
- `migrations/` - Database schema changes
- `requirements.txt` - Project dependencies

## Technologies
- Python
- SQLAlchemy or similar ORM framework
- SQL databases

## Getting Started
1. Install dependencies: `pip install -r requirements.txt`
2. Configure database connection settings
3. Run migrations to set up the database
4. Execute scripts to test ORM functionality
