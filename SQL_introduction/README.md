# SQL Introduction

This directory contains introductory SQL exercises and projects from the Holberton School higher-level programming curriculum.

## Concepts

- Introduction to SQL
- Database basics
- SQL queries and commands
- Data manipulation and retrieval

## Files

| File | Description |
|------|-------------|
| `0-list_databases.sql` | List all databases |
| `1-create_database.sql` | Create a new database |
| `2-delete_database.sql` | Delete a database |
| `3-list_tables.sql` | List all tables in a database |
| `4-full_description.sql` | Show table structure |
| `5-first_table.sql` | Create your first table |
| `6-list_values.sql` | List all rows in a table |
| `7-insert_value.sql` | Insert data into a table |
| `8-count_records.sql` | Count records in a table |
| `9-full_creation.sql` | Create and populate a table |
| `10-top_band.sql` | Query and filter data |

## Learning Objectives

- Understand relational databases
- Write basic SQL queries
- Create and manage tables
- Insert, update, and retrieve data
