
# SQL_more_queries

This directory contains advanced SQL query exercises and projects from the Holberton School higher-level programming curriculum.

## Contents

- **Queries**: Complex SELECT statements with JOINs, subqueries, and aggregations
- **Database Design**: Table creation, constraints, and relationships
- **Data Manipulation**: INSERT, UPDATE, DELETE operations
- **Optimization**: Efficient query patterns and best practices

## Learning Objectives

- Master advanced SQL syntax and operations
- Work with multiple tables and joins
- Understand database normalization
- Write efficient and optimized queries

## Requirements

- MySQL or compatible SQL database
- Basic SQL knowledge

## Usage

1. Clone the repository
2. Navigate to the `SQL_more_queries` directory
3. Use the provided SQL files to practice and learn advanced SQL concepts

## Files

- `0-privileges.sql` - User privilege management
- `1-create_user.sql` - Create database users
- `2-create_read_user.sql` - Create read-only users
- `3-force_integer_value.sql` - Integer constraints
- `4-create_database.sql` - Database creation
- `5-unique_id.sql` - Unique constraints
- `6-states.sql` - States table queries
- `7-cities.sql` - Cities and states joins
- `8-cities_of_california_subquery.sql` - Subquery examples
- `9-genres_by_show.sql` - Genre aggregation
- `10-rank_shows_by_rating.sql` - Rating rankings
