# JavaScript Warm Up

Welcome to the JavaScript Warm Up project! This directory contains introductory JavaScript exercises designed to help you build foundational programming skills.

## Learning Objectives

- Understand JavaScript syntax and basic concepts
- Practice writing simple scripts
- Learn variable declaration and data types
- Work with functions and control flow
- Get comfortable with the JavaScript runtime environment

## Project Structure

```
javascript-warm_up/
├── README.md
└── [exercise files]
```

## Getting Started

1. Navigate to this directory
2. Create JavaScript files for each exercise
3. Run scripts using Node.js:
    ```bash
    node script_name.js
    ```
